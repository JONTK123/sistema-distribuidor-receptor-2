# sistema-distribuidor-receptor-2
Implementação de Merge Sort distribuído em Java. O módulo Distribuidor envia fragmentos de vetores a Receptores, que usam threads para ordenar dados paralelamente. As partes são devolvidas e intercaladas. Utiliza TCP/IP, serialização de objetos e inclui medição de tempos para comparar a eficiência do modelo distribuído vs. sequencial.
